Passo 1: Baixar o arquivo zipado do repositório do Yazi;
Passo 2: Descompactar o arquivo baixado com o comando "unzip";
Passo 3: Criar uma pasta uma pasta dentro do $HOME para ter os conteúdos do arquivo baixado;
Passo 4: Copiar os arquivos da pasta descompactada para a pasta criada no passo 3;
Passo 5: Appendar a pasta criada no passo 3 no $PATH com o comando export PATH=$PATH:$HOME/yazi.
